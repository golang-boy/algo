# 树与图


1. 链表是特殊的树
2. 树是特殊的图
  n个点，n-1条边连通的无向图，这是树
  n个点，n条边的连通无向图，为基环树


怎么存储图？

   * 邻接矩阵

       二维表格，适用于稠密图,  空间复杂度O(n^2)
       无向图时，对称矩阵，可以只存一半
       稀疏图，空间浪费严重

       直接开二维数组  graph[m][n]

       新增边: 直接graph[m][n] = 1

   * 出边数组

       表示某个点可以到达哪些点
       一个n的数组，每个数组中存储一个变长数组
       适用于稀疏图

       [][]int graph[n] 或者 map[int][]int graph[n]

       新增边：graph[m] = append(graph[m], n)


   * 邻接表

       与出边数组类似，只是后面挂的是链表

       新增边：graph[m].Add(n)


图的遍历

   * 深度优先搜索

     划分连通块
     找环

     图遍历时，需要判重，走过的节点不用走

   * 广度优先遍历

     拓扑排序
